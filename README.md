## 🚀 快速开始

### 1. 下载仓库

```bash
git clone https://github.com/Destined-at-Dawn/obsidian-ai-toolkit.git
```

或直接下载ZIP：右上角 Code → Download ZIP

### 2. 阅读指南

打开 `Obsidian+AI新手入门指南.md`，跟着做：

1. **安装篇**（10分钟）— Obsidian + 三大AI插件
2. **创建仓库**（5分钟）— 你的第一个知识库
3. **AI辅助写作**（5分钟）— 用AI生成结构化笔记
4. **多源采集**（10分钟）— Reddit/YouTube/X内容一键抓取
5. **进阶玩法**（选做）— 本地LLM、自动化工作流

### 3. 解压工具包

```bash
unzip niuma-starter-kit-v2.zip -d starter-kit/
```

里面有：
- 📝 笔记模板（科研笔记、读书笔记、项目复盘等）
- 🤖 AI prompt模板（写作、分析、总结、翻译等）
- ⚡ 工作流自动化脚本

## 💡 本指南是怎么写出来的？

指南的第七章完整记录了**调研过程**：

1. 从Reddit r/ObsidianMD 采集社区真实反馈（100+讨论帖）
2. 从YouTube搜索最新教程（观看3个高赞视频并提取要点）
3. 从X/Twitter搜索实时讨论（验证信息时效性）
4. 交叉验证数据，筛选出2026年真正有效的方案

**这个过程本身就是范例**——你可以照抄这个方法去调研任何主题。
